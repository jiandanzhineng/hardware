## 文档结构
1. 概述：说明 BLE 操作的总体思路与适用范围，具体设备属性请参考 `docs/device_api_reference.md`。
2. 连接与发现：如何连接、发现服务 `0x00FF` 与特征列表，说明设备名来源（BLUFI 设置）。
3. 属性映射：通过特征的 User Description 获取属性名，以此建立“特征 ↔ 属性名”的映射。
4. 读/写/订阅：
- 读：读取特征值
- 写：按类型编码（int/float/string）写入特征值
- 订阅：写入 CCCD 启用通知，设备在属性变化时主动推送
5. 模式切换：通过特征 `0xFF02` 写入 0x01/0x00 切换 BLE/WiFi 模式。
6. 数据编码规则：int 小端、float 原样二进制、string 原文，长度限制简述。
7. 注意事项：默认不启用 BLE SMP 配对；广播/连接流程由 BLUFI 管理；属性类型与边界请参考设备 API 文档。
8. 参考与代码定位：列出关键实现位置，便于开发者快速跳转。
