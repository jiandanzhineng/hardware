# 设备连接方式说明

本文档说明当前项目支持的设备连接方式。

## 1. WiFi 连接方式

支持设备连接到特定 WiFi 或默认 WiFi。

### WiFi 网络配置

*   **默认 WiFi**
    *   SSID: `easysmart`
    *   密码: `11111111`
*   **特定 WiFi**
    *   需通过 BluFi（蓝牙配网）进行配置。

### 服务器连接方式

*   **网关模式（热点方式）**
    *   连接对象：手机热点或电脑热点。
    *   限制：连接数量受限（一般 8 个以内）。
    *   说明：此时网关即为热点设备。

*   **mDNS 方式（easysmart.local）**
    *   连接对象：本地路由器。
    *   地址：通过 mDNS 发布的地址 `easysmart.local` 访问。
    *   优势：支持更多设备连接（数量仅受路由器限制）。
    *   说明：此方式下网关和服务器地址可能不一致。

## 2. BLE 方式连接

*   支持通过蓝牙（BLE）直接连接并控制设备。
